# SenegalCheckers
It is a checkers game for Android. Senegal checkers were used as a reference, but game
 rules were modified. You can read the rules
 [here](https://github.com/ppzhuk/SenegalCheckers/blob/master/rules.txt).

#### Requrinments
* Android 4.4
